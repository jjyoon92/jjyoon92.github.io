도서 리뷰 모바일 앱

2023.08(1인 프로젝트)

- BloC 패턴을 통해 일관적이고 효율적인 상태 관리를 하여 앱의 아키텍처를 깔끔하게 구성했습니다.
- 네이버북 API를 활용하여 실시간 도서 검색 기능을 제공함으로써 사용자들은 쉽게 원하는 도서를 찾을 수 있으며, 사용자에게 더 나은 검색 경험을 제공합니다.
- Firebase Authentication을 사용하여 구글 로그인을 제공하고, 2차적으로 회원 가입까지 완료 인증이되면 서비스를 이용할 수 있습니다.
- Firebase Database를 이용하여 사용자의 회원 정보와 도서 리뷰 및 별점을 저장하고 동기화함으로써, 앱은 실시간으로 업데이트 됩니다.

Flutter를 본격적으로 사용한 프로젝트이며 BloC패턴에 대한 깊이 있는 공부를 하게되어 좋은 경험이라고 생각합니다. 차후에 사용자 간의 정보 교환을 위해 리뷰 목록을 제공하거나 실시간 채팅 기능등을 추가 개발 할 예정입니다.

주요 기능

도서 검색, 별점 등록 및 리뷰 작성, Firebase연동(프로필 사진, 별점, 리뷰 저장), BloC패턴 적용. 사용자 관리

기차 예매 모바일 앱

2023.05~2023.07(팀 프로젝트)

SRT앱을 클론코딩한 기차 예매 서비스 모바일 앱입니다. 어떤 주제를 선정하면 좋을지를 고민하다가 공공데이터API를 적극적으로 활용하면서 실력향상을 위해 도전적인 서비스를 만들어보고 싶어 시작하게 되었습니다.
Figma를 사용하여 와이어프레임과, 프로토타입을 제작하고 Android Native로 개발하였습니다. 서버는 SpringBoot를 사용하였으며 MariaDB를 데이터베이스로 사용하였습니다.
주 목표로 삼았던 회원인증과 노선 조회, 예매 및 환불처리를 개발 완료하였으며 이를 개발하는 동안 많은 것들을 배웠지만 특히 의존성 주입을 적극 사용해 볼 수 있었던 기회였습니다.

주요 기능

실제 SRT 노선데이터를 바탕으로 열차 운행 정보 제공. 세션에 기반한 회원인증을 통해 열차 예매 및 예매 내역 확인, 취소 기능을 제공.

향수 추천 MBTI 테스트(WEB)

2022.05(2인 프로젝트)

기획자 친구와 함께 협업하여 만든 MBTI 테스트 입니다.
기존의 다른 MBTI 테스트와의 차별점을 두고, 보다 명확하게 사용자의 성향에 맞춰 제품을 추천하기위해 32가지의 결과를 만들었습니다.
별도의 서버 개발없이 Firebase Database를 통해 테스트 횟수나 결과 빈도등을 수집하였으며, 도메인 구매 없이 웹서비스 배포를 위해 Netlify를 이용하여 배포를 하였습니다. Kakao, Facebook, Twitter 공유 API를 사용하여 SNS 공유가 가능하고, clipboard.js 를 사용하여 현재 보고 있는 결과페이지의 주소를 그대로 복사 할 수 있습니다. 비록 철지난 MBTI 테스트지만 처음으로 직접 구상하고 개발하여 웹 서비스 배포를 해보았다는 것에 의미를 가진 프로젝트입니다.

주요 기능

선택지에 따른 다양한 결과 제공, SNS 공유기능, 결과화면 링크 복사, Firebase 데이터 저장 및 수집,
